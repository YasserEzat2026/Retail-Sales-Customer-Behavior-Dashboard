# Interactive Retail Sales & Customer Behavior Dashboard
## Project Overview
An advanced, fully interactive Microsoft Excel Business Intelligence solution designed to analyze retail sales performance, monitor distribution channels, and track customer satisfaction metrics. This dynamic dashboard provides corporate stakeholders with immediate visual insights into regional revenue trends, product category movement, and comprehensive consumer demographic behavioral patterns.

## Key Features & Insights
* **Comprehensive Sales & Slicing Controls:** Built a robust data architecture filtered seamlessly via dynamic Slicers tracking Managers, Employee Sales Metrics, Product Categories, and absolute Temporal splits (Years, Quarters, Months).
* **Customer Demographics & Segmentation:** Developed tailored categorical breakdowns evaluating revenue distributions across Customer Types (Loyal, New, Retail) and Gender metrics (Male vs. Female).
* **Geographical & Operational Distribution:** Visualized localized store performance ranking sales distribution across major commercial hubs and cities (Riyadh, Tabuk, Medina, Jeddah).
* **CRM Satisfaction Analysis:** Embedded key performance evaluation metrics tracking Customer Satisfaction rankings (High, Low, Ok, Very High, Very Low) to optimize retention management.

## Tools & Technologies Used
* **Data Warehousing & Processing:** Microsoft Excel utilizing core aggregation functions and automated tables.
* **Analytical Modeling:** Pivot Tables and conditional calculations for structural summary aggregation.
* **UI/UX Visualization:** Advanced Excel charting, dynamic Slicers, and custom color-palette application for executive presentation.
